# 🎸 FretMaster

> 吉他指板视觉化练习工具 | Guitar Fretboard Visualization Practice Tool

[![Next.js](https://img.shields.io/badge/Next.js-16-black?logo=next.js)](https://nextjs.org/)
[![React](https://img.shields.io/badge/React-19-blue?logo=react)](https://react.dev/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?logo=typescript)](https://www.typescriptlang.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 🌟 功能特点

- **音程练习** - 识别和演奏各种音程
- **和弦练习** - 练习和弦构成音，支持多种和弦类型
- **音阶练习** - 熟悉各种音阶在指板上的位置
- **找音练习** - 提高对指板的熟悉程度
- **和弦进行** - 练习常见和弦进行
- **调音器** - 内置吉他调音器

## 🌐 国际化支持

- 🇨🇳 中文
- 🇺🇸 English

## 🛠️ 技术栈

- **框架**: [Next.js 16](https://nextjs.org/) (App Router)
- **UI**: [React 19](https://react.dev/) + [Tailwind CSS](https://tailwindcss.com/)
- **组件**: [shadcn/ui](https://ui.shadcn.com/)
- **语言**: [TypeScript](https://www.typescriptlang.org/)
- **状态管理**: Zustand
- **构建工具**: Turbopack

## 📦 安装

```bash
# 克隆仓库
git clone https://github.com/lerk-dev/fretmaster.git
cd fretmaster

# 安装依赖
npm install
# 或
pnpm install
```

## 🚀 运行

```bash
# 开发模式
npm run dev

# 构建生产版本
npm run build

# 运行生产版本
npm run start
```

## 📖 使用说明

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `ESC` | 停止练习 / 退出全屏 |
| `1-5` | 切换练习标签页 |
| `F` | 切换全屏模式 |
| `M` | 切换麦克风 |
| `S` | 打开设置 |
| `P` | 开始/停止练习 |
| `H` | 显示/隐藏快捷键帮助 |
| `空格` / `→` | 下一题 |
| `↑` | 显示指板 |
| `↓` | 隐藏指板 / 下一题 |

## 🎯 练习模式

### 音程练习
练习识别和演奏各种音程，从小二度到纯八度及以上。

### 和弦练习
选择和弦类型和练习等级，练习和弦构成音在指板上的位置。

### 音阶练习
选择音阶类型和调性，练习音阶音符在指板上的位置。

### 找音练习
随机显示目标音符，在指板上找到对应位置。

## 📱 响应式设计

支持桌面端和移动端，自适应不同屏幕尺寸。
