# 2026-04-30 工作日志

## FretMaster vs Solo 全面对比分析

完成了 FretMaster (v0.2.22) 与 Solo (v2.4.0) 的全面对比分析，生成文档 `fretmaster-vs-solo-comparison.md`。

### 核心发现

**FretMaster 优势（6项）：**
1. 跨平台 Web + Tauri 桌面
2. 中英文双语
3. Rust 音频滤波链（高通/低通/陷波）— Solo 没有
4. 统计系统（按天/类型/详细追踪）— Solo 没有
5. 音高算法可选（standard / solo）
6. 噪音抑制、智能练习建议

**Solo 优势（6项）：**
1. Android Oboe 原生音频低延迟
2. 自实现 269KB FloatFFT.java（三种算法可选）
3. 完整的自定义调弦系统（CRUD + 导出导入）
4. 设备校准系统（按设备 ID 存储 + 微调）
5. 音频设备管理（热插拔/USB/多声道）
6. Google Play 商业成熟度

**关键差距（Solo有的）：**
- 自定义调弦系统（最重要缺失）
- 水平指板模式
- 设备校准系统
- 音频设备热插拔
- 和弦符号字体
- PostHog 分析

**FretMaster page.tsx 达 709KB，急需代码拆分。**
