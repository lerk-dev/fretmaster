## 一、功能模块介绍

### 1.1 调音器 (Tuner)

**功能说明**：内置吉他调音器，帮助你在练习前确保吉他音准正确。

**操作步骤**：
1. 点击顶部导航栏的 🎵 调音器图标
2. 点击麦克风图标启用音频输入
3. 拨动吉他弦，观察指针位置
4. 指针指向中央表示音准正确
5. 偏左表示音偏低，偏右表示音偏高

**应用场景**：
- 每次练习前检查吉他音准
- 学习调音技巧
- 训练音高辨别能力

---

### 1.2 指板可视化 (Fretboard)

**功能说明**：吉他指板的可视化展示，显示所有品格和琴弦的位置。

**显示内容**：
- 6根琴弦（从低音E到高音E）
- 12个品格位置
- 音符名称标注
- 音程关系显示

**操作步骤**：
1. 在主界面查看指板显示
2. 点击设置调整显示方式（中文/英文/简写/爵士）
3. 练习时观察目标音符在指板上的位置

---

### 1.3 音程练习 (Interval Practice)

**功能说明**：练习识别和演奏各种音程，从小二度到纯八度及以上。

**练习模式**：
- **找根音模式**：先找到根音，再找目标音程
- **直接模式**：直接找到目标音程音

**操作步骤**：
1. 选择"音程练习"标签
2. 选择要练习的音程类型（如纯四度、纯五度）
3. 选择根音模式或直接模式
4. 点击"开始练习"
5. 根据提示在指板上点击正确的音符

**音程类型**：
| 音程 | 半音数 | 说明 |
|------|--------|------|
| 小二度 | 1 | 紧张感 |
| 大二度 | 2 | 自然音阶 |
| 小三度 | 3 | 悲伤感 |
| 大三度 | 4 | 明亮感 |
| 纯四度 | 5 | 稳定感 |
| 三全音 | 6 | 不稳定 |
| 纯五度 | 7 | 和谐 |
| 小六度 | 8 | 忧郁 |
| 大六度 | 9 | 温暖 |
| 小七度 | 10 | 爵士感 |
| 大七度 | 11 | 引导音 |
| 纯八度 | 12 | 完全和谐 |

---

### 1.4 和弦练习 (Chord Practice)

**功能说明**：练习和弦构成音，支持多种和弦类型和练习等级。

**和弦类型**：
- 三和弦：大三、小三、增三、减三
- 七和弦：大七、小七、属七、半减七、减七
- 扩展和弦：九和弦、十一和弦、十三和弦

**练习等级**：
| 等级 | 说明 | 适合人群 |
|------|------|----------|
| 原位三和弦(1,3,5) | 基础三和弦 | 初学者 |
| 三音+五音+七音(3,5,7) | 转位练习 | 进阶者 |
| 随机转位三和弦 | 灵活应用 | 进阶者 |
| 原位七和弦(1,3,5,7) | 七和弦基础 | 进阶者 |
| 随机转位七和弦 | 高级转位 | 高级玩家 |
| 扩展练习(1,3,5,7,1) | 完整练习 | 高级玩家 |

**操作步骤**：
1. 选择"和弦练习"标签
2. 选择和弦类型（如大三和弦、属七和弦）
3. 选择练习等级
4. 选择演奏顺序（上行/下行/随机）
5. 点击"开始练习"
6. 按顺序点击和弦构成音

---

### 1.5 音阶练习 (Scale Practice)

**功能说明**：熟悉各种音阶在指板上的位置，支持多种指法模式。

**音阶分类**：

| 分类 | 包含音阶 | 特点 |
|------|----------|------|
| 基础音阶 | 大调、自然小调、和声小调、旋律小调 | 最常用 |
| 教会调式 | 伊奥尼亚、多里安、弗里几亚等 | 爵士基础 |
| 五声音阶 | 大调五声、小调五声 | 流行/摇滚 |
| Bebop音阶 | Bebop大调、Bebop小调等 | 爵士演奏 |
| 爵士音阶 | 布鲁斯、Altered、Whole Tone等 | 高级爵士 |
| 异域音阶 | 西班牙、阿拉伯、印度等 | 特殊风格 |
| 对称音阶 | 半音阶、增音阶、三全音 | 现代音乐 |

**练习序列**：
- **1→1**：从根音到根音（基础）
- **3→3**：从三音到三音（转位）
- **5→5**：从五音到五音（转位）
- **7→7**：从七音到七音（高级）
- **随机**：随机琶音模式

**操作步骤**：
1. 选择"音阶练习"标签
2. 选择音阶分类和具体音阶
3. 选择练习序列
4. 选择演奏方向（上行/下行/随机）
5. 点击"开始练习"
6. 按序列顺序点击音阶音符

---

### 1.6 找音练习 (Note Finding)

**功能说明**：随机显示目标音符，在指板上找到对应位置，提高指板熟悉度。

**练习设置**：
- 练习时长：5分钟、10分钟、15分钟、30分钟
- 目标音符：随机生成
- 答案验证：实时反馈

**操作步骤**：
1. 选择"找音练习"标签
2. 选择练习时长
3. 点击"开始练习"
4. 看到目标音符后，在指板上点击正确位置
5. 答对后自动进入下一题

---

### 1.7 和弦进行 (Chord Progression)

**功能说明**：练习常见和弦进行，支持iReal Pro格式导入。

**内置进行**：
- II-V-I 进行（爵士基础）
- I-V-vi-IV 进行（流行）
- 蓝调进行
- 自定义进行

**iReal Pro导入**：
1. 复制iReal Pro格式的和弦进行文本
2. 粘贴到导入框
3. 点击"导入"
4. 系统自动解析并生成练习

**操作步骤**：
1. 选择"和弦进行"标签
2. 选择内置进行或导入自定义进行
3. 设置速度和调性
4. 点击"开始练习"
5. 按顺序演奏和弦

## 二、设置区域详解

### 2.1 练习设置 (Practice Settings)

#### 练习时间
- **作用**：设置单次练习的时长
- **默认值**：0（无限制）
- **可选值**：0秒、30秒、60秒、120秒、300秒
- **修改方法**：拖动滑块或输入数值
- **影响**：练习倒计时结束后自动停止

#### 冷却时间
- **作用**：答对一题后的等待时间，防止误触
- **默认值**：启用，500毫秒
- **可选值**：0-2000毫秒
- **修改方法**：
  1. 启用/禁用：点击开关
  2. 调整时长：拖动滑块
- **影响**：
  - 启用：答对后需等待才能进入下一题
  - 禁用：答对后立即进入下一题

---

### 2.2 节拍器设置 (Metronome Settings)

#### 启用节拍器
- **作用**：开启/关闭节拍器
- **默认值**：关闭
- **修改方法**：点击开关

#### 节拍器速度 (BPM)
- **作用**：设置节拍器速度
- **默认值**：120 BPM
- **取值范围**：40-240 BPM
- **修改方法**：拖动滑块
- **应用场景**：
  - 慢速(60-80)：初学练习
  - 中速(100-120)：日常练习
  - 快速(140+)：速度训练

#### 节拍器声音
- **作用**：是否播放节拍器声音
- **默认值**：开启
- **修改方法**：点击开关

#### 节拍器闪光
- **作用**：视觉节拍提示
- **默认值**：关闭
- **修改方法**：点击开关
- **适用场景**：静音练习时使用

---

### 2.3 音频输入设置 (Audio Input Settings)

#### 音频输入设备
- **作用**：选择麦克风设备
- **默认值**：系统默认设备
- **修改方法**：
  1. 点击下拉菜单
  2. 选择可用设备
- **注意**：首次使用需要授权麦克风权限

#### 输入增益
- **作用**：调整输入音量大小
- **默认值**：100%
- **取值范围**：0-200%
- **修改方法**：拖动滑块
- **调整建议**：
  - 信号太弱：提高增益
  - 信号过载（削波）：降低增益

#### 启用麦克风
- **作用**：开启/关闭音频输入
- **默认值**：关闭
- **修改方法**：点击开关
- **显示**：开启后显示当前使用的音频处理模式（AudioWorklet 或 ScriptProcessor）

---

### 2.4 音高检测参数 (Pitch Detection)

这些参数用于调整音高检测算法的敏感度。

#### 灵敏度 (Sensitivity)
- **作用**：控制音高检测的敏感度
- **默认值**：0.5
- **取值范围**：0.0-1.0
- **影响**：
  - 高灵敏度：更容易检测到音高，但可能误检
  - 低灵敏度：更严格，减少误检但可能漏检
- **调整建议**：
  - 环境安静：0.6-0.8
  - 环境嘈杂：0.3-0.5

#### 置信度阈值 (Confidence Threshold)
- **作用**：检测结果的可信度门槛
- **默认值**：0.8
- **取值范围**：0.0-1.0
- **影响**：
  - 高阈值：只接受高置信度的结果，更准确但更严格
  - 低阈值：接受更多结果，更宽松但可能不准确
- **调整建议**：
  - 追求准确：0.8-0.9
  - 快速响应：0.6-0.7

#### 噪音等级 (Noise Level)
- **作用**：背景噪音的阈值
- **默认值**：0.1
- **取值范围**：0.0-1.0
- **影响**：高于此值的信号才被视为有效音频

#### 最小音量 (Minimum Volume)
- **作用**：检测的最小音量门槛
- **默认值**：0.01
- **取值范围**：0.0-0.1
- **影响**：过滤掉太弱的声音

#### 缓冲区大小 (Buffer Size)
- **作用**：音频处理缓冲区大小
- **默认值**：4096
- **可选值**：2048、4096、8192、16384
- **影响**：
  - 小缓冲区：低延迟，但可能不稳定
  - 大缓冲区：更稳定，但延迟增加

#### 谐波权重 (Harmonic Weights)
- **作用**：谐波分析的权重
- **默认值**：0.5
- **取值范围**：0.0-1.0
- **影响**：调整对谐波的敏感度

#### 零交叉阈值 (Zero Crossing Threshold)
- **作用**：用于区分浊音和清音
- **默认值**：0.05
- **取值范围**：0.0-0.2
- **影响**：调整对噪音的过滤程度

---

### 2.5 MIDI设置 (MIDI Settings)

#### 启用MIDI
- **作用**：开启/关闭MIDI设备支持

#### MIDI设备选择

### 2.6 显示设置 (Display Settings)

#### 语言设置
- **作用**：切换界面语言
- **默认值**：中文
- **可选值**：中文、English
- **修改方法**：点击对应语言按钮

#### 主题设置
- **作用**：切换深色/浅色主题
- **默认值**：跟随系统
- **可选值**：深色、浅色
- **修改方法**：点击主题切换按钮

#### 指板显示方式
- **作用**：设置音符显示格式
- **默认值**：中文
- **可选值**：
  - 中文：哆、来、咪、发、嗦、啦、西
  - 英文：C、D、E、F、G、A、B
  - 英文简写：C、D、E、F、G、A、B（简写形式）
  - 爵士：Do、Re、Mi、Fa、Sol、La、Ti
- **修改方法**：点击设置中的下拉菜单

### 2.7 设置管理

#### 导出设置
- **作用**：将当前设置保存为文件
- **使用方法**：点击"导出设置"按钮
- **文件格式**：JSON

#### 导入设置
- **作用**：从文件加载设置
- **使用方法**：
  1. 点击"导入设置"按钮
  2. 选择之前导出的JSON文件
  3. 确认导入

#### 重置设置
- **作用**：恢复所有设置为默认值
- **使用方法**：点击"重置设置"按钮
- **注意**：此操作不可撤销

## 四、快捷键

### 4.1 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `ESC` | 停止练习 / 退出全屏 | 立即停止当前练习 |
| `1-5` | 切换练习标签页 | 1:调音器, 2:音程, 3:和弦, 4:音阶, 5:找音 |
| `F` | 切换全屏模式 | 最大化练习区域 |
| `M` | 切换麦克风 | 快速启用/禁用音频输入 |
| `S` | 打开设置 | 打开设置面板 |
| `P` | 开始/停止练习 | 快速开始或停止练习 |
| `H` | 显示/隐藏快捷键帮助 | 查看所有快捷键 |

### 4.2 练习模式快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `空格` / `→` | 下一题 | 跳过当前题目 |
| `↑` | 显示指板 | 显示答案提示 |
| `↓` | 隐藏指板 / 下一题 | 隐藏提示或进入下一题 |

### 4.3 快捷键使用技巧

1. **练习流程**：
   - 按 `P` 开始练习
   - 答题后按 `空格` 进入下一题
   - 遇到困难按 `↑` 查看提示
   - 完成练习按 `ESC` 停止

2. **设备调试**：
   - 按 `M` 快速启用麦克风
   - 按 `S` 打开设置调整参数
   - 按 `H` 查看快捷键帮助

## 五、常见问题

### Q1: 麦克风没有反应怎么办？

**解决方法**：
1. 检查浏览器是否授予麦克风权限
2. 确认选择了正确的音频输入设备
3. 调整输入增益到合适值（100-150%）
4. 检查麦克风是否正常工作

### Q2: 音高检测不准确怎么办？

**解决方法**：
1. 调整灵敏度参数（建议0.6-0.8）
2. 降低置信度阈值（建议0.7-0.8）
3. 增大缓冲区大小（建议8192）
4. 确保环境安静，减少背景噪音

### Q3: 练习时声音有延迟怎么办？

**解决方法**：
1. 减小缓冲区大小（建议2048或4096）
2. 关闭不必要的后台程序
3. 使用有线耳机代替蓝牙设备
4. 检查电脑性能是否足够

### Q4: 如何导入自定义和弦进行？

**步骤**：
1. 准备iReal Pro格式的文本（如 `C7 | F7 | G7 | C7`）
2. 打开"和弦进行"标签
3. 点击"导入"按钮
4. 粘贴文本并确认

### Q5: 手机使用时声音会回放怎么办？

**说明**：这是音频反馈问题，已在最新版本修复。

**如果仍有问题**：
1. 使用耳机练习
2. 降低设备音量
3. 确保使用最新版本

### Q6: 如何备份我的设置？

**步骤**：
1. 打开设置面板
2. 滚动到"设置管理"区域
3. 点击"导出设置"按钮
4. 保存JSON文件到安全位置

## 六、进阶技巧

### 6.1 高效练习方法

1. **每日练习计划**：
   - 调音：2分钟
   - 找音练习：5分钟
   - 音程练习：10分钟
   - 和弦练习：10分钟
   - 音阶练习：10分钟

2. **循序渐进**：
   - 第1-2周：找音练习 + 基础音程
   - 第3-4周：加入三和弦练习
